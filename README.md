# 여성 전용 가치관 기반 모임 앱 — MVP 정리

> 지금까지의 논의를 한 문서로 정리한 사업 컨셉 요약본입니다.

---

## 1. 한 줄 정의

> **"가치관이 맞는 모임장과 모임을 추천해주는, 20-30대 여성 전용 라이프스테이지 기반 모임 앱"**

기존 모임 앱(소모임·문토)이 "카테고리로 모임을 찾는" 곳이라면, 우리는 **"가치관으로 사람(모임장)을 찾고, 그 모임장의 모임에 들어가는"** 곳.

---

## 2. 타겟 고객

### 핵심 페르소나
- **20대 후반 ~ 30대 여성 직장인**
- 코어 타겟: **30대 비혼·비출산 여성** (시장 빠르게 성장 중, 기존 서비스 거의 없음)
- 확장 타겟: 20대 후반 사회초년생, 30대 워킹맘, 여성 프리랜서·창업가

### 페인포인트
- 기존 모임 앱(소모임·문토)은 남초 + 술자리·등산 위주 → 여성에게 불편
- 결혼·출산 안 한다는 이유로 어색해지는 경험
- 정치·종교·페미니즘 등 가치관 충돌로 모임 분위기 망가짐
- 30대가 되니 친구 사귈 자리가 없음 (이직·이사·결혼으로 관계 리셋)
- 익명 커뮤니티(속닥 등)는 1020 위주라 결이 안 맞음

---

## 3. 시장 분석

### 경쟁 환경
| 영역 | 대표 서비스 | 우리와 관계 |
|------|-----------|-----------|
| 1020 여성 익명 커뮤니티 | 속닥 (60만+) | 결이 다름 (익명 수다 vs 실명 모임) |
| 5070 여성 시니어 | 시놀 | 타겟 다름 |
| 2030 여성 멤버십 | 헤이조이스(컬리 인수 후 침체), 빌라선샤인(소규모) | **사실상 빈자리** |
| 남녀혼성 모임 앱 | 소모임, 문토, 트레바리 | 차별화 가능 (여성전용+가치관) |
| 글로벌 레퍼런스 | Bumble BFF, Peanut, Chief | 한국에 동급 없음 |

### 핵심 인사이트
- **"여성 커뮤니티 앱은 많다"는 착각**: 1020 익명 수다 시장만 포화. 2030 라이프스테이지 기반 모임 시장은 비어있음.
- **소모임이 남초인 이유**: 여성 수요 부재가 아니라, 남성 친화적 콘텐츠(술·등산) 위주로 구조가 짜여 있어서 발생한 악순환.
- **마켓컬리의 헤이조이스 인수**: 2030 여성 커뮤니티가 커머스로 연결되면 큰 가치가 있다는 시장 검증.

## 4. 핵심 차별화 — 가치관 매칭 엔진

### 4-1. 가치관 프로필 (온보딩)
가입 시 3~5분 내 완료되는 30~50개 질문으로 가치관 프로필 생성.

**질문 카테고리:**
- **라이프스타일**: 결혼·출산관, 직업관, 거주형태
- **인간관계**: 친밀도 스타일, 대화 선호, 정치·사회 이슈 관심도
- **가치관**: 페미니즘 입장, 종교, 정치 성향, 환경·동물권 관심도
- **취향**: MBTI, 술/담배, 운동 빈도, 영화·음악 취향

**필수 원칙**: 모든 민감 항목은 "비공개" 옵션 제공.

### 4-2. 모임장 중심 매칭 (핵심 IP)
참가자 전원 매칭 대신 **"모임장의 가치관"** 과 매칭.

- 모임장이 모임 분위기를 100% 결정함
- 계산이 가벼우면서도 정확함
- 같은 모임장에 끌린 사람끼리 자연스럽게 결이 맞음
- 모임장 팬덤이 생기면 자연스러운 락인

**매칭 가중치:**
- 라이프스타일·가치관 = 50%
- 인간관계 스타일 = 20%
- 취향 = 20%
- 거리·연령 = 10%

### 4-3. 모임장 검증·교육
- 모임장 신청 시 영상 인터뷰
- 가치관 진정성 체크 (자기소개 영상 제출)
- 모임 운영 가이드 제공
- 모임 후 참가자 평가 → 모임장 등급 시스템
- 우수 모임장 인센티브

## 5. MVP 핵심 기능

### Phase 1 (출시 시점)
1. **가치관 프로필 온보딩** (30개 질문 / 3분)
2. **모임장 가치관 매칭 추천** (호환성 점수 표시)
3. **모임 둘러보기 + 신청** (피드 형태)
4. **여성 본인 인증** (휴대폰 + 추가 검증)
5. **모임 채팅방** (참가자끼리 소통)
6. **모임 후기·평점** (참가자가 모임장 평가)

### Phase 2 (3~6개월 후)
- 같은 모임장 팔로우 기능
- 정기 모임(시즌제) 운영 시스템
- 추천 알고리즘 고도화

### Phase 3 (6~12개월 후)
- 오프라인 모임 수수료 모델 도입
- 여성 타겟 브랜드 제휴
- 프리미엄 기능 (선택적)

## 6. 초기 시드 모임 컨셉 예시

런칭 시 운영자가 직접 큐레이션할 시드 모임 (각 1~2개 모임장 섭외):

1. **30대 비혼 여성의 자산관리 스터디** (8주, 격주 모임)
2. **워킹맘의 새벽 6시 러닝 클럽** (월 4회)
3. **여성 프리랜서의 사이드프로젝트 모임** (3개월 시즌)
4. **30대 비혼 여성의 1인 미식 클럽** (월 2회)
5. **여성 글쓰기·에세이 모임** (8주)
6. **30대 여성의 미니멀 라이프 스터디** (4주)
7. **여성 직장인 커리어 고민 라운드테이블** (월 1회)
8. **여성 여행 동행 매칭 모임** (시즌제)
9. **20대 후반 이직 준비 스터디** (8주)
10. **여성 비건·환경 라이프 모임** (월 2회)

## 7. 포지셔닝 & 브랜딩

### Do
- "결이 맞는 사람들의 모임"
- "비슷한 처지의 여성들과 깊이 연결"
- "안전하고 의미 있는 시간"
- "내 또래 또래 여성 모임"

### Don't
- "여성 전용 모임 앱" (직설적, 페미니즘 프레임 위험)
- "남자 없는 안전한 모임" (부정적 어워더리스)
- "여자들만의 비밀 모임" (폐쇄적 이미지)

→ "여성전용"은 결과일 뿐, 메시지의 핵심이 아니어야 함.

---

## 8. 수익 모델 (단계적)

### MVP 단계 (0~12개월)
- **무료** — 사용자 확보가 최우선
- 자체 운영비로 버티며 PMF 검증

### Phase 2 (1~2년차)
- **오프라인 모임 수수료** (참가비의 10~15%)
- 트레바리·문토 모델

### Phase 3 (2년차 이후)
- **여성 타겟 브랜드 제휴·커머스** (뷰티·웰니스·여행)
- **프리미엄 기능** (VIP 모임, 우선 매칭 등)
- **B2B 데이터·인사이트** (트렌드 리포트, 포커스 그룹)

---

## 9. 초기 모객 전략 (광고비 0원으로 1,000명까지)

### 0~2개월: 시드 100명
- 창업자가 직접 30~50명 1:1 인터뷰
- 지인 + 지인의 지인 네트워크
- 초대제 베타 운영 (희소성 활용)

### 2~4개월: 마이크로 인플루언서
- 팔로워 1만~5만 30대 여성 인플루언서 3~5명 협업
- 30대 비혼 라이프 / 1인가구 / 워킹맘 채널
- 인스타 릴스·유튜브 숏츠 콘텐츠

### 4~6개월: 콘텐츠·PR
- 브런치·뉴스레터 에세이
- 첫 오프라인 모임 후기 콘텐츠화
- 시사인·여성신문 PR 시도

### 6개월+: 검증된 PMF로 유료 광고 시작
- 인스타·페이스북·틱톡 광고

## 10. 주요 리스크 & 대응

| 리스크 | 대응 방안 |
|--------|---------|
| "여성전용"의 정치적 프레임 | 라이프스테이지 중심 메시지, "전용" 강조 X |
| 모임장 품질 편차 | 검증·교육·평가 시스템 |
| 가치관 매칭의 부정확성 | 알고리즘 + 휴먼 큐레이션 병행 |
| 가입 온보딩 이탈 (질문 과다) | 30개 이내, 3분 이내, 비공개 옵션 |
| 초기 사용자 부족(닭-달걀 문제) | 시드 모임장 운영자가 직접 섭외 |
| 페미니즘 이슈로 인한 분란 | 가치관 매칭으로 사전 필터링 |
